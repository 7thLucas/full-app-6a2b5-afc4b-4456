# Product Overview — MushroomAI

## Product Name
**MushroomAI** *(working name — to be confirmed with founder)*

## Tagline
*"Know before you eat."*

## What It Is
MushroomAI is a conversational AI web application that identifies mushrooms from photos or text descriptions. Users chat naturally with the AI to learn whether a mushroom is safe to eat, what species it is, and what they need to know before foraging. It is built for non-experts and designed for use in the field.

## The Problem
Mushroom misidentification is a serious and growing risk. Thousands of poisonings occur annually — some fatal — because foragers, hikers, and curious cooks cannot reliably distinguish edible from toxic species. Existing tools (printed field guides, online forums) are slow, hard to carry, and require expert knowledge to interpret correctly. There is no fast, reliable, always-available identification resource that a non-expert can trust in the field.

## The Solution
A mobile-first AI chat interface where users upload a mushroom photo or describe what they see, and receive:
- A species identification with a confidence level
- A clear safety rating (Safe / Caution / Toxic / Unknown)
- A species profile with distinguishing features
- Look-alike warnings for dangerous similar species
- Natural language follow-up conversation

## Target Users
1. **Casual foragers** — picking wild mushrooms for personal or family cooking
2. **Hikers & outdoor enthusiasts** — encountering mushrooms on trail and wanting to know if they're safe
3. **Mycology hobbyists** — learning to identify species for knowledge and interest
4. **Home cooks & foodies** — sourcing or curious about wild and specialty mushrooms

## Core Features (MVP)
1. **Photo Identification** — upload a mushroom photo; AI identifies species with confidence level
2. **Natural Language Chat** — ask follow-up questions in plain language ("Is this safe raw?" / "What does it taste like?")
3. **Safety Rating** — explicit edibility status: Safe / Caution / Toxic / Unknown — never ambiguous
4. **Species Profile** — scientific name, common name, habitat, season, distinguishing features
5. **Look-alike Warnings** — highlights dangerous species the mushroom could be confused with
6. **Identification History** — saved log of past identifications per user session

## Brand & Tone
- **Tone**: Trustworthy, clear, safety-first. Knowledgeable but never condescending. Calm reassurance paired with honest uncertainty disclosures.
- **Visual Identity**: Earthy, nature-inspired. Deep forest greens, warm ambers, cream whites. Premium and approachable.
- **Key principle**: Never overstate AI confidence. When uncertain, say so explicitly and recommend expert verification.

## Strategic Principles
1. **Safety first** — Uncertainty is always disclosed; look-alike warnings are always shown
2. **Non-expert accessible** — No mycology jargon without a plain-language explanation
3. **Speed in the field** — Results in seconds, mobile-first UX
4. **Transparent AI** — Confidence scores and alternative species shown; never a black-box verdict

## MVP Scope
- Web-based chat UI (mobile-optimised)
- Photo upload + text description input
- AI identification with safety rating and species profile
- Look-alike warnings
- Session-based identification history

## Out of Scope (Post-MVP)
- Community / social features
- Expert human verification layer
- Offline mode
- Subscription / monetisation tier
- Push notifications
